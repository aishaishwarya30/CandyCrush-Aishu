# CandyCrush-Aishu

A fun and interactive **Candy Crush-style game** built using **HTML, CSS, and JavaScript**.
This project demonstrates grid-based logic, drag-and-drop functionality, and dynamic score tracking.

---

## 🚀 Features

* 🎮 Interactive candy matching gameplay
* 🔄 Restart game functionality
* ⏱️ Timer-based gameplay *(if added)*
* ⭐ Score tracking system
* 🎨 Improved UI and responsive design

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## 📌 How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Run the `index.html` file in your browser

---

## 💡 Project Description

This game is inspired by the classic Candy Crush concept where players match candies in a grid to score points.
The project focuses on implementing game logic such as matching patterns, updating scores, and handling user interactions.
